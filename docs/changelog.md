# 更新日志

本文件记录 Uaha 项目的所有重要变更。

格式遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [v0.1.0-beta] - 2026-02-11

### 🎉 首个公开 Beta 版本

这是 Uaha 项目的首次公开发布，提供基础的模型管理和 API 服务管理功能。

### ✅ 已完成功能

#### 核心功能
- **模型管理**
  - 浏览可用的 AI 模型列表
  - 下载模型，支持实时进度显示
  - 删除已安装的模型
  - 模型状态监控（available/downloading/installed/error）

- **API 服务管理**
  - 创建新的服务实例
  - 启动/停止服务
  - 查看服务运行状态
  - 服务资源监控（内存、GPU、延迟）

- **配置管理**
  - Aha 引擎路径配置
  - 模型下载目录设置
  - 下载重试次数设置
  - 主题切换（亮色/深色）
  - 外观自定义（主色调、圆角、字体大小）
  - 自动启动和自动更新选项

#### 技术实现
- 基于 Wails v3 的跨平台架构
- Vue 3 + Composition API 构建现代化 UI
- Pinia 状态管理
- Tailwind CSS 样式系统
- 实时事件推送机制

### 🖥️ 平台支持

| 平台 | 状态 | 备注 |
|------|------|------|
| macOS | ✅ 完成 | Intel 和 Apple Silicon 兼容性测试通过 |
| Windows | 🚧 开发中 | 核心功能可用，待完善兼容性测试 |
| Linux | 🚧 开发中 | 核心功能可用，待完善兼容性测试 |


### ⚠️ 已知限制

1. **平台兼容性**
   - Windows 和 Linux 版本尚未完成完整的兼容性测试
   - 部分平台特定功能可能存在 Bug

2. **国际化**
   - 国际化（i18n）功能待实现
   - 当前仅支持中文界面

3. **系统 Tray**
   - 系统 Tray（托盘）功能待设置与实现
   - 最小化到托盘功能暂不可用

4. **高级配置**
   - 部分高级配置选项需要通过命令行操作
   - GUI 配置界面待完善

## 版本规范说明

Uaha 遵循语义化版本规范 (Semver)：`主版本号.次版本号.修订号`

- **主版本号 (Major)**: 不兼容的 API 变更
- **次版本号 (Minor)**: 向下兼容的功能性新增
- **修订号 (Patch)**: 向下兼容的问题修正

附加标签：
- `-alpha`: 内部测试版本
- `-beta`: 公开测试版本
- `-rc`: 发布候选版本
